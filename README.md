# rock paper scissors++

This code is a big bang theory twist on the classic game, rock paper scissors, now with two additional things to pick, lizard and spock
- rock: wins against scissors and lizard
- paper: wins against spock and rock
- scissors: wins against paper and lizard
- lizard: wins against paper and spock
- spock: wins against rock and scissors

## features
this code features a win and a loss counter, and you can play as long as you want
